  • B2C 단건 구매 — Stripe 결제 연동, 라이선스 유형별 가격 차등, 구매 후 워터마크 적용 다운로드

  • 포렌식 워터마크 — IMATAG API 연동, 비가시적 워터마크 삽입으로 불법 유통 추적
  • DRM 스트리밍 — Pallycon 연동, 영상 콘텐츠 암호화 스트리밍, 무단 복제 방지
  • 관리자 패널 — 콘텐츠·사용자·구독·매출 통합 관리, 대시보드 통계

제안 포인트

본 프로젝트는 NFT 마켓플레이스 + 디지털 콘텐츠 DRM + B2B/B2C 이커머스가 결합된 복합 플랫폼입니다. 다음과 같은 강점을 기반으로 성공적 구축을 제안합니다:

  • NFT 마켓플레이스 직접 구축 경험 — Ethereum 기반 NFT 거래 플랫폼(마켓플레이스, 토큰 스왑, 에어드롭)을 8개월간 풀스택으로 개발한 경험이 있으며, ethers.js·스마트 컨트랙트·IPFS 연동 전반을 직접 구현
  • 동일 기술 스택 숙련도 — Next.js + Node.js/Express + PostgreSQL 조합으로 다수 프로젝트를 수행하여 기술 스택 학습 비용 없이 즉시 개발 착수 가능
  • 대규모 B2B 포탈 구축 경험 — 50+ 페이지, 200+ API, 80+ 엔티티 규모의 B2B SaaS 플랫폼을 14개월간 구축한 경험으로 관리자 패널·결제·구독 관리 영역의 검증된 설계 역량 보유
  • 외부 API 연동 전문성 — Stripe 결제, KYC 인증, AI API 등 15+ 외부 API 연동 경험으로 IMATAG·Pallycon 등 신규 API 통합도 효율적으로 진행

PRD (Product Requirements Document)

프로젝트 요구사항을 정의하고 기능 범위를 명확히 합니다.

1. 프로젝트 배경 및 목적

한국 전통 예술 콘텐츠(이미지·음원·영상)를 디지털 라이선스로 거래하는 마켓플레이스를 구축합니다. 개인 구매자(B2C)와 기관 구독자(B2B) 모두를 지원하는 단일 카탈로그 기반 포털로, NFT 라이선스 증명·포렌식 워터마크·DRM 스트리밍을 통해 콘텐츠 보호와 거래 투명성을 확보합니다.

3. 기능 요구사항

3.1 콘텐츠 카탈로그

  • [필수] 이미지·음원·영상 통합 카탈로그 브라우징
  • [필수] 카테고리·태그·키워드 기반 검색 및 필터링
  • [필수] 콘텐츠 미리보기 (저해상도 이미지, 음원 샘플, 영상 프리뷰)
  • [필수] 콘텐츠 상세 페이지 (메타데이터, 라이선스 유형, 가격)
  • [선택] 추천 콘텐츠 및 인기 콘텐츠 섹션

3.2 B2C 단건 구매

  • [필수] Stripe 결제 연동 (신용카드/간편결제)
  • [필수] 라이선스 유형별 가격 차등 (개인용/상업용)
  • [필수] 구매 완료 시 포렌식 워터마크 적용 후 다운로드
  • [필수] 구매 이력 및 다운로드 기록 관리

3.3 B2B 기관 구독

  • [필수] 월/연 구독 플랜 관리
  • [필수] Stripe 인보이스 기반 구독 결제
  • [필수] 기관별 사용량 추적 및 보고
  • [필수] 구독 라이선스 범위 관리 (다운로드 수 제한 등)

3.4 NFT 라이선스 증명

  • [필수] Polygon 네트워크 ERC-721 NFT 민팅
  • [필수] IPFS(web3.storage) 메타데이터 영구 보관
  • [필수] 라이선스 소유권 온체인 검증
  • [필수] NFT 라이선스 조회 및 이력 확인

3.5 콘텐츠 보호 (워터마크/DRM)

  • [필수] IMATAG API 연동 — 비가시적 포렌식 워터마크 삽입
  • [필수] Pallycon DRM — 영상 콘텐츠 암호화 스트리밍
  • [선택] 불법 유통 탐지 및 알림

3.6 관리자 패널

  • [필수] 콘텐츠 관리 (등록/수정/삭제/승인)
  • [필수] 사용자 관리 (B2C/B2B/관리자)
  • [필수] 구독 관리 및 인보이스 조회
  • [필수] 매출 대시보드 및 통계
  • [필수] NFT 민팅 관리

5. 기술 제약사항

  • 프론트엔드: Next.js 14 (App Router) + Tailwind CSS
  • 백엔드: Node.js + Express / Next.js API Routes
  • 데이터베이스: PostgreSQL + Prisma ORM
  • 파일 저장소: S3 호환 오브젝트 스토리지 (AWS S3 또는 MinIO)
  • 결제: Stripe (단건 + 구독 인보이스)
  • 워터마킹: IMATAG API
  • DRM: Pallycon
  • 블록체인: Polygon + ethers.js
  • IPFS: web3.storage
